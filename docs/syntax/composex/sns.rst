======
x-sns
======

Syntax
======

.. code-block:: yaml
    :caption: x-sns syntax reference


    x-sns:
      Topics:
        TopicA:
          Properties: {}
          Settings: {}
          Services: []
      Subscriptions:
        SubscriptionA:
          Properties: {}
          Settings: {}
          Topics: []


.. warning::

At this current version, **Subscriptions** are not supported.

===========

Refer to `AWS SNS Topic Documentation`_ for SNS Topics
